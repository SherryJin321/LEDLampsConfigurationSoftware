# LEDLampsConfigurationSoftware

## 1. 软件名称 <br>
Airsafe 灯具参数配置软件<br>

## 2. 软件版本<br>
### 2.1 版本 V1.0 <br>
#### 2.1.1 软件类型 <br>
Visual Studio WPF 软件
#### 2.1.2 软件功能<br>
  * 可选择工厂模式或开发者模式，配置Airsafe灯具参数。<br>
  * 可查询Airsafe 灯具版本信息和状态信息。<br>
  * 想进入开发者模式，需先进行用户登录。<br>
